# ルーティング + 自己変更プロトコル

## 配置先の判断基準
| 条件 | 配置先 |
|------|--------|
| 絶対禁止（deny/hooks） | settings.json（P0） |
| 全セッション共通の身分・価値観 | CLAUDE.md（P1） |
| 短い行動規範（20行以下） | rules/（P2） |
| 詳細な手順書 | skills/（P3） |
| プロジェクト固有 | {project}/CLAUDE.md（P4） |
| 揮発メモ | CLAUDE.local.md（P5） |

## 自己変更時の必須手順
1. 変更案と配置先を人間に提示
2. 人間が認証パスワードを入力して承認
3. 認証成功後に変更を実行
4. CLAUDE.local.md に変更内容と理由を記録
5. **settings.json は絶対に変更禁止**
